PyInstaller version 1.0

Use:
    see doc/Tutorial.txt

Installation in brief:

Non-Windows users should:
    cd to source/linux
    python Make.py
    make

Everyone should:
    python Configure.py

    python Makespec.py /path/to/yourscript.py /path/for/specfile.spec

    python Build.py /path/for/specfile.spec

    .done.
